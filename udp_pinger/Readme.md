## Сборка:
```
cd build && cmake .. && make && cd ../
```

## Запуск клиента:
```
./build/udp_pinger_client 127.0.0.1
```

## Запуск сервера:
```
./build/udp_pinger_server
```

## Описание алгоритма:
1. Расчет RTT:
- Перед вызовом `sendto()` программа фиксирует текущее время с помощью `gettimeofday()`.
- Сразу после успешного возврата из `recvfrom()` фиксируется время прибытия.
- *Формула*: RTT = (Tend.sec - Tstart.sec) + (Tend.usec - Tstart.usec)/1.000.000
- Результат выводится в секундах с высокой точностью (миллисекунды).

2. Обработка тайм-аутов:
- Используется системный вызов `setsockopt()` с флагом `SO_RCVTIMEO`.
- Сокету назначается структура `timeval` со значением 1 секунда.
- Если сервер не отвечает в течение этого времени, вызов `recvfrom()` прерывается и возвращает -1.
- Программа проверяет код ошибки (errno). Если он равен `EAGAIN` или `EWOULDBLOCK`, выводится сообщение "Request timed out".

Ответ от сервера: Ping 2 1703258124, RTT = 0.000115 сек
...
